문제

현재 미세먼지 농도를 입력하면 그 결과를 바탕으로 좋음, 보통, 나쁨, 매우나쁨을 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 참고로, 현재 미세먼지 농도( ㎍/m³)가 30이하이면 '좋음', 30 - 80이면 '보통', 80 - 150이면 '나쁨', 151이상이면 '매우나쁨'입니다.
0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : 60
미세먼지 농도는 대부분의 지역이 '보통' 수준을 보이고 있다


0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : 14
미세먼지 농도는 대부분의 지역이 '좋음' 수준을 보이고 있다


0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : 100
미세먼지 농도는 대부분의 지역이 '나쁨' 수준을 보이고 있다

코드 빈칸 채우기

#include <stdio.h>  
 
int main() 
{ 
	int fineDust = 0; 
 
	printf("0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : " ); 
	scanf( "%d", &fineDust ); 
 

	 ( fineDust  30 )								 
	{ 
		printf( "미세먼지 농도는 대부분의 지역이 '좋음' 수준을 보이고 있다 " ); 
	} 
	 ( ( 30 < fineDust )  ( fineDust  80 ) )				 
	{ 
		printf( "미세먼지 농도는 대부분의 지역이 '보통' 수준을 보이고 있다 " ); 
	} 
	 ( ( 80 < fineDust )  ( fineDust  150 ) )				 
	{ 
		printf( "미세먼지 농도는 대부분의 지역이 '나쁨' 수준을 보이고 있다 " ); 
	} 
	 										 
	{ 
		printf( "미세먼지 농도는 대부분의 지역이 '매우나쁨' 수준을 보이고 있다 " ); 
	} 
	 
	return 0; 
}
