문제

8부터 15사이의 10진수를 입력받아 2진수로 변환하여 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
8부터 15사이의 10진수를 입력하세요 : 11
10진수 11은 2진수 1011입니다.


8부터 15사이의 10진수를 입력하세요 : 10
10진수 10은 2진수 1010입니다.


8부터 15사이의 10진수를 입력하세요 : 8
10진수 8은 2진수 1000입니다.

코드 빈칸 채우기

#include <stdio.h> 
 
int main() { 
	int binary = 0, binary1000 = 0, binary100 = 0, binary10 = 0, binary1 = 0; 
	int decimal = 0, temp = 0; 
 
	printf( "8부터 15사이의 10진수를 입력하세요 : " ); 
	scanf( "%d", &decimal ); 
	temp = decimal;  
 
	binary1000 = temp  ;					 
	temp = temp  ;					 
 
	binary100 = temp  ;					 
	temp = temp  ;					 
 
	binary10 = temp  ;					 
	temp = temp  ;					 
 
	binary1 = temp; 
 
	binary = binary1000 * 1000 + binary100 * 100 + binary10 * 10 + binary1;	 
 
	printf( "10진수 %d은 2진수 %d입니다.\n", decimal, binary ); 
 
	return 0; 
}
