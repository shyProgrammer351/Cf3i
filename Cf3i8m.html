문제

1000부터 1111사이의 2진수를 입력받아 10진수로 변환하여 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
1000부터 1111사이의 2진수를 입력하세요: 1011
2진수 1011은 10진수 11입니다.

프로그램 코드

	#include <stdio.h>
	
1	int main() {
2		int binary = 0, binary1000 = 0, binary100 = 0, binary10 = 0, binary1 = 0, decimal = 0;
	
3		printf( "1000부터 1111사이의 2진수를 입력하세요: " );
4		scanf( "%d", &binary );
	
5		binary1000 = binary / 1000;							
	
6		binary100 = ( binary % 1000 ) / 100;							
	
7		binary10 = ( binary % 100 ) / 10;							
	
8		binary1 = ( binary % 10 );
	
9		decimal = binary1000 * 8 + binary100 * 4 + binary10 * 2 + binary1; 	
	
10		printf( "2진수 %d은 10진수 %d입니다.\n", binary, decimal );
	
11		return 0;
	}
