문제

1000부터 1111사이의 2진수를 입력받아 10진수로 변환하여 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
1000부터 1111사이의 2진수를 입력하세요: 1011
2진수 1011은 10진수 11입니다. 


1000부터 1111사이의 2진수를 입력하세요: 1010
2진수 1010은 10진수 10입니다. 


1000부터 1111사이의 2진수를 입력하세요: 1111
2진수 1111은 10진수 15입니다.

코드 빈칸 채우기

#include <stdio.h> 
	 
int main() { 
	( "1000부터 1111사이의 2진수를 입력하세요: " ); 
	 binary = 0; 
	( "", &binary ); 
	 
	 binary1000 = binary  1000; 
	 
	 binary100 = ( binary  1000 )  100; 
	 
	 binary10 = ( binary  100 )  10; 
	 
	 binary1 = ( binary  10 ); 
	 
	 decimal = binary1000 *  + binary100 *  + binary10 *  + binary1; 
	 
	( "2진수 은 10진수 입니다.", binary, decimal ); 
	 
	return 0; 
}
