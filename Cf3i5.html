문제

알파벳 소문자를 입력받아 대문자를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
알파벳 소문자 한 글자를 입력하세요: k
소문자 k의 대문자는 K입니다^^


알파벳 소문자 한 글자를 입력하세요: j
소문자 j의 대문자는 J입니다^^


알파벳 소문자 한 글자를 입력하세요: h
소문자 h의 대문자는 H입니다^^

알고리즘

프로그램 시작 
   알파벳 소문자를 입력받기 
   알파벳 소문자를 대문자로 변환하기 
   입력값 출력하기 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/Alphabet.c

	#include <stdio.h>

 

	// 프로그램 시작 

1	int main() {

		char lowercase = '\0';

2		char uppercase = '\0';

	 

		// 알파벳 소문자를 입력받기 

3		printf("알파벳 소문자 한 글자를 입력하세요: " );

4		scanf( "%c", &lowercase );

	 

		// 알파벳 소문자를 대문자로 변환하기 

5		uppercase = lowercase - 'a' + 'A';

	 

		// 입력값 출력하기 

6		printf("소문자 %c의 대문자는 %c입니다^^", lowercase, uppercase );

	 

		// 프로그램 종료 

7		return 0;

	}
