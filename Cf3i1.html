문제

변수에 값을 저장할 때 발생하는 overflow와 underflow를 실습하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
max     =  2147483647
max + 1 = -2147483648 ( overflow발생 ) 
min     = -2147483648
min - 1 =  2147483647 ( underflow발생 )

알고리즘

프로그램 시작 
   int의 최대값에 1을 더하면 overflow 발생 
   int의 최소값에 1을 빼면 underflow 발생 
   overflow와 underflow를 실습 결과 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/OverflowUnderflow.c

	#include <stdio.h>

	 

	// 프로그램 시작 

1	int main() {

		// int의 최대값에 1을 더하면 overflow 발생 

2		int max = +2147483647;

3		int overflow = max + 1;

	 

		// int의 최소값에 1을 빼면 underflow 발생 

4		int min = -2147483648;

5		int underflow = min - 1;

	 

		// overflow와 underflow를 실습 결과 출력  

6		printf( "max     =  %d\n", max );

7		printf( "max + 1 = %d ( overflow 발생 )\n", overflow );

8		printf( "min     = %d\n", min );

9		printf( "min - 1 =  %d ( underflow 발생 )\n", underflow );

	 

		// 프로그램 종료 

10		return 0;

	}
