문제

알파벳 소문자를 입력받아 대문자를 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
알파벳 소문자 한 글자를 입력하세요: k
소문자 k의 대문자는 K입니다^^

1 2와 5 6 사이의 실행순서(2개) 및 실행상태

3  scanf( "%c", &lowercase );
k
          main()
lowercase 

4  char uppercase = (char)( (int) lowercase - (int) 'a' + (int) 'A' );
          main()
lowercase 'k'
uppercase


프로그램 코드

	
	#include <stdio.h>

1	int main() {
2		printf( "알파벳 소문자 한 글자를 입력하세요: " );
		char lowercase = '\0';
3		scanf( "%c", &lowercase );
	
4		char uppercase = (char)( (int) lowercase - (int) 'a' + (int) 'A' );
	
5		printf( "소문자 %c의 대문자는 %c입니다^^", lowercase, uppercase );	
	
6		return 0;
	}
