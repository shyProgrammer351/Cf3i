문제

8부터 15사이의 10진수를 입력받아 2진수로 변환하여 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
8부터 15사이의 10진수를 입력하세요 : 11
10진수 11은 2진수 1011입니다.


8부터 15사이의 10진수를 입력하세요 : 10
10진수 10은 2진수 1010입니다.


8부터 15사이의 10진수를 입력하세요 : 8
10진수 8은 2진수 1000입니다.

알고리즘

프로그램 시작 
   8부터 15사이의 10진수를 입력 
   10진수에서 2진수의 각 자리수를 추출 
      2진수 1000 = 2^3 = 8의 자리수 
      2진수 100 = 2^2 = 4의 자리수 
      2진수 10 = 2^1 = 2의 자리수 
      2진수 1 = 2^0 = 1의 자리수 
   2진수 각 자리수의 값을 바탕으로 10진수를 2진수로 변환 
   10진수를 2진수로 변환한 결과를 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/DecimalToBinary.c

	#include <stdio.h>

	 

	// 프로그램 시작 

1	int main() {

		// 변수 선언 및 초기화

2		int decimal = 0, binary1000 = 0, binary100 = 0, binary10 = 0, binary1 = 0, binary = 0;

	 

		// 8부터 15사이의 10진수를 입력 

3		printf( "8부터 15사이의 10진수를 입력하세요 : " );

4		scanf( "%d", &decimal );

	 

		// 10진수에서 2진수의 각 자리수를 추출 

		// 2진수 1000 = 2^3 = 8의 자리수 

5		binary1000 = decimal / 8;

	 

		// 2진수 100 = 2^2 = 4의 자리수 

6		binary100 = ( decimal % 8 ) / 4;

	 

		// 2진수 10 = 2^1 = 2의 자리수 

7		binary10 = ( decimal % 4 ) / 2;

	 

		// 2진수 1 = 2^0 = 1의 자리수 

8		binary1 = decimal % 2;

	 

		// 2진수 각 자리수의 값을 바탕으로 10진수를 2진수로 변환 

9		binary = binary1000 * 1000 + binary100 * 100 + binary10 * 10 + binary1;

	 

		// 10진수를 2진수로 변환한 결과를 출력 

10		printf( "10진수 %d은 2진수 %d입니다.\n", decimal, binary );

	 

		// 프로그램 종료 

11		return 0;

	}
