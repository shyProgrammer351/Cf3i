문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 -2, -1, 0, 1, 2 등의 정수 자료형  

 pre  = ++2 는 2+1( 3 ) 실행후 pre = 3  

 post = 2++ 는 post = 2 실행후 2+1( 3 )  

 pre  = --2 는 2-1( 1 ) 실행후 pre = 1  

 post = 2-- 는 post = 2 실행후 2-1( 1 )
